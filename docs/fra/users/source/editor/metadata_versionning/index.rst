Gestion des versions des métadonnées
====================================

Il est possible d'activer le suivi des versions pour une fiche de métadonnées. 
L'activation de la gestion des versions d'une fiche est accessible depuis la page 
de résultat ou dans la page de consultation d'une fiche 
dans le menu **Autres actions**. A chaque modification, la fiche est mise à jour 
dans le système de gestion des versions (ie. Subversion).

Pour le moment, pour consulter l'historique, il est nécessaire de mettre en place un client spécifique pour Subversion (cf. http://trac.osgeo.org/geonetwork/wiki/metadatachanges).
